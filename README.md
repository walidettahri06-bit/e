# ESM - Elevators Sahara Morocco

## Project Info

**Website**: ESM - Expert en ascenseurs et BTP au Maroc

## Comment exécuter ce projet ?

Suivez ces étapes :

```sh
# Step 1: Installer les dépendances.
npm install

# Step 2: Démarrer le serveur de développement.
npm run dev
```

## Technologies utilisées

Ce projet est construit avec :

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Comment déployer ce projet ?

Build le projet pour la production :

```
sh
npm run build
```
